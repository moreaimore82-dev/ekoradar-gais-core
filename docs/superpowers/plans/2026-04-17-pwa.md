# PWA Desteği — Uygulama Planı

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** EkoRadar'ı Android ve diğer cihazlara kurulabilir bir Progressive Web App haline getirmek; sağ alt köşede floating install butonu, kurulum sonrası buton kayboluyor.

**Architecture:** Yeni npm bağımlılığı yok. Statik `public/` dosyaları (manifest, SW, SVG ikonlar) + `index.html` güncellemesi + `App.tsx`'e install butonu state/efekt/JSX eklenmesi.

**Tech Stack:** Vite + React + TypeScript + Tailwind + Framer Motion (`motion/react`). Service Worker: vanilla JS. İkonlar: SVG.

## Task 5: Install Butonu (App.tsx)

**Files:**
- Modify: `src/App.tsx`

- [ ] **Adım 1: `BeforeInstallPromptEvent` tip tanımını ekle**

`src/App.tsx` dosyasının en başına (import'lardan önce) ekle:

```tsx
interface BeforeInstallPromptEvent extends Event {
  prompt(): Promise<void>;
  userChoice: Promise<{ outcome: 'accepted' | 'dismissed' }>;
}
```

- [ ] **Adım 2: Install state'lerini ekle**

`App()` fonksiyonu içindeki mevcut state listesine (örn. `passwordError` state'inden sonra) ekle:

```tsx
const [installPrompt, setInstallPrompt] = useState<BeforeInstallPromptEvent | null>(null);
const [isInstalled, setIsInstalled] = useState(() => localStorage.getItem('pwa_installed') === 'true');
```

- [ ] **Adım 3: `beforeinstallprompt` ve `appinstalled` event listener'larını ekle**

Mevcut `useEffect`'lerin yanına yeni bir `useEffect` ekle:

```tsx
useEffect(() => {
  const handleBeforeInstall = (e: Event) => {
    e.preventDefault();
    setInstallPrompt(e as BeforeInstallPromptEvent);
  };
  const handleInstalled = () => {
    setIsInstalled(true);
    setInstallPrompt(null);
    localStorage.setItem('pwa_installed', 'true');
  };
  window.addEventListener('beforeinstallprompt', handleBeforeInstall);
  window.addEventListener('appinstalled', handleInstalled);
  return () => {
    window.removeEventListener('beforeinstallprompt', handleBeforeInstall);
    window.removeEventListener('appinstalled', handleInstalled);
  };
}, []);
```

- [ ] **Adım 4: `handleInstall` fonksiyonunu ekle**

Diğer handler fonksiyonlarının yanına (örn. `handlePasswordSubmit`'in altına) ekle:

```tsx
const handleInstall = async () => {
  if (!installPrompt) return;
  await installPrompt.prompt();
  const { outcome } = await installPrompt.userChoice;
  if (outcome === 'accepted') {
    setIsInstalled(true);
    localStorage.setItem('pwa_installed', 'true');
  }
  setInstallPrompt(null);
};
```

- [ ] **Adım 5: Floating install butonu JSX'ini ekle**

`src/App.tsx`'in en sonundaki kapanış `</div>` etiketinden hemen önce (Arşiv modalının `</AnimatePresence>` kapanışından sonra) ekle:

```tsx
{/* PWA Install Button */}
<AnimatePresence>
  {installPrompt && !isInstalled && (
    <motion.button
      initial={{ opacity: 0, y: 20, scale: 0.9 }}
      animate={{ opacity: 1, y: 0, scale: 1 }}
      exit={{ opacity: 0, y: 20, scale: 0.9 }}
      transition={{ type: 'spring', stiffness: 300, damping: 25 }}
      onClick={handleInstall}
      className="fixed bottom-6 right-6 z-50 flex items-center gap-2 bg-zinc-900 dark:bg-white text-white dark:text-zinc-900 px-4 py-3 rounded-2xl shadow-2xl border border-zinc-700 dark:border-zinc-200 hover:scale-105 active:scale-95 transition-transform"
    >
      <Download className="w-4 h-4" />
      <span className="text-sm font-bold tracking-tight">Uygulamayı Yükle</span>
    </motion.button>
  )}
</AnimatePresence>
```

Not: `Download` ikonu `lucide-react`'tan zaten import edilmiş (`src/App.tsx:25`).
